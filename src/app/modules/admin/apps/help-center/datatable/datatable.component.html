Admin Interface Overview

The admin interface allows for managing agencies and agents, changing security settings, and viewing agency-related information.

Create Agency

In this section, admins can create a new agency by providing required information such as agency name, code, contact email, phone number, address, region, and city. Once completed, the user can cancel or submit the form using the "Create Agency" button.

Create Agent

This section allows the creation of a new agent under an agency. The form includes fields for the agent's username, password, email, and phone number. Once completed, the user can either cancel or click "Save" to add the agent.

Associate Agent to Agency

This section provides the ability to associate an existing agent with an agency by selecting both from dropdowns. After selecting, the user can save or cancel the association.

Admin Security (Change Password)

Admins can change their password in this section by entering the current and new passwords. A password generator option is also available if needed. The admin can either save the new password or cancel the process.

Generate Random Password for Admins

This section allows the admin to generate a random password for other admins who may have forgotten their current password. The admin can select from the dropdown and click "Generate" to produce a new password.

Admin Team (Agency Information)

This section displays a table with agency-related information such as agency name, code, email, city, region, associated agent, status, and an option to edit agency details. There are filters for region and city, as well as pagination options for better navigation.
